# Step 1. Launch New Scale Computing HyperCore Cluster Wizard


To launch the New Scale Computing HyperCore Cluster wizard, do the following:

1. In the Veeam Backup & Replication console, open the Backup Infrastructure view.
2. In the inventory pane, select Managed Servers.
3. On the ribbon, click Add Server.
4. In the Add Server window, select Virtualization Platforms.
5. In the Virtualization Platforms window, select Scale Computing HyperCore.

Alternatively, you can do the following:

1. In the Veeam Backup & Replication console, open the Inventory view.
2. In the inventory pane, select Virtual Infrastructure.
3. On the ribbon, click Add Server.
4. In the Add Server window, select Virtualization Platforms.
5. In the Virtualization Platforms window, select Scale Computing HyperCore.
