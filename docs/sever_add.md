# Adding Scale Computing HyperCore Cluster to Backup Infrastructure


To add a Scale Computing HyperCore cluster to the backup infrastructure, do the following:

1. [Launch the New Scale Computing HyperCore Cluster wizard](server_add_launch.md).
2. [Specify the Scale Computing HyperCore cluster domain name or IP address](server_add_name.md).
3. [Enter credentials to access the Scale Computing HyperCore cluster](server_add_credentials.md).
4. [Apply Scale Computing HyperCore cluster settings](server_add_apply.md).
5. [Finish working with the wizard](server_add_summary.md).
