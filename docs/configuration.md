# Configuring Veeam Plug-in for Scale Computing HyperCore


To start working with Veeam Plug-in for Scale Computing HyperCore, perform a number of steps for its configuration:

1. [Configure backup repositories](configure_repository.md) where Veeam Plug-in for Scale Computing HyperCore will store backups of Scale Computing HyperCore VMs.
2. [Connect to Scale Computing HyperCore clusters](sever_add.md) that administer Scale Computing HyperCore resources you want to protect.
3. [Deploy workers](workers.md) that will transfer backup traffic.
4. [[Optional] Configure email settings and notifications](general_settings.md).
