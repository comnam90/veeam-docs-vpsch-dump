# Removing Scale Computing HyperCore Cluster


If you do not want to protect resources managed by the connected Scale Computing HyperCore cluster anymore, you can remove it from the backup infrastructure.

To remove the Scale Computing HyperCore cluster from the backup infrastructure:

1. Open the Backup Infrastructure view.
2. In the inventory pane, select Managed Servers > Scale Computing HyperCore.
3. In the working area, select the necessary cluster and click Remove Cluster on the ribbon.

Alternatively, right-click the cluster and select Remove.
