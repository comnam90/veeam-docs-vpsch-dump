# Creating Backup Jobs


To create a backup job, do the following:

1. [Check prerequisites and limitations](backup_job_create_prerequisites.md).
2. [Launch the New Backup Job wizard](backup_job_create_launch.md).
3. [Specify a job name and description](backup_job_create_general_settings.md).
4. [Selects VMs to back up](backup_job_create_assign_vms.md).
5. [Specify a backup repository and configure backup settings](backup_job_create_destination.md).
6. [Create a schedule for the backup job](backup_job_create_schedule.md).
7. [Finish working with the wizard](backup_job_create_summary.md).
